## 🏥 Hospital-Wards-Fall-and-Injury-Management-Power-BI-Dashboard

This project focuses on analysing patient falls and injury incidents within a hospital setting to support patient safety and quality improvement. Using real-world healthcare concepts, the dashboard provides insights into fall trends, risk factors, and injury outcomes to help healthcare teams make data-driven decisions.

### 🔍 Project Objectives

* Identify patterns and trends in patient falls
* Analyse high-risk patient groups and contributing factors
* Monitor injury severity and incident frequency
* Support early intervention and prevention strategies

### 📊 Key Features

* Interactive Power BI dashboard with KPI tracking
* Analysis of falls by age group, location, and risk category
* Injury severity breakdown and trend analysis
* Visual insights to support clinical decision-making

### 🛠️ Tools & Technologies

* Power BI (Data Modelling, DAX, Dashboards)
* SQL Server (Data extraction and transformation)
* Excel (Data cleaning and preparation)

### 💡 Key Insights

* Identified high-risk patient groups with increased fall incidents
* Highlighted peak periods and locations for falls
* Provided actionable insights for improving patient safety measures

### 🎯 Outcome

This project demonstrates the use of data analytics in healthcare to improve patient safety, reduce fall incidents, and support evidence-based decision-making within clinical environments.
